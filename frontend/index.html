MODEL ARCHITECTURE
Ensemble composition and feature engineering pipeline
Isolation Forest
weight 35%
Global anomaly detection via random recursive partitioning. Anomalies isolated in fewer splits. O(n log n). 200 trees, max_samples=256.
Gaussian Mixture Model
weight 30%
8-component probabilistic density model. Anomaly score = −log P(x). Trained with EM. Identifies transactions outside learned behaviour clusters.
Local Outlier Factor
weight 20%
Density-based local anomaly detection. LOF > 1 means sparse region relative to neighbours. Catches contextual anomalies missed by global methods. k=20.
Statistical Z-score
weight 15%
Chi-squared Mahalanobis distance across feature space. Threshold at 97.5th percentile. Per-user rolling baseline via 50-transaction window.
